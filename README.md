# F2E Bus

## Demo 網址
https://yileeya.github.io/f2eBus/

## 作品說明
- 參加[2021 THE F2E 前端修煉精神時光屋](https://2021.thef2e.com/)的第三週作品
- API 由[交通部 TDX](https://tdx.transportdata.tw/api-service/swagger)提供

## 功能
- 查看縣市中的所有公車路線
- 查看某路線的到站狀況，包含地圖模式與路線軸模式
- 查看某路線的發車時間表


## 資料夾說明
- src/main.js - 主程式進入點
- src/components - Vue Components
- src/router - Vue-router 路由檔案
- src/store - Vuex 存存 modules，及 api
- src/assets - 圖檔放置位置
- public - 接入起始檔案
